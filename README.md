# 🛡️ CyberSecurity Journey

> *"Every expert was once a beginner."*

Welcome to my **CyberSecurity Journey**! 👋

This repository documents my learning path from **beginner to advanced** in cybersecurity. Here, I share my study notes, hands-on labs, TryHackMe write-ups, Linux commands, networking concepts, web security topics, ethical hacking techniques, and useful cybersecurity resources.

My goal is not only to learn cybersecurity but also to create a well-organized knowledge base that can help other beginners understand complex concepts in simple English.

---

# 🎯 Goals

* Build a strong foundation in cybersecurity.
* Learn networking, Linux, and operating systems.
* Master ethical hacking and penetration testing.
* Practice on real-world labs and challenges.
* Document everything I learn.
* Help other beginners by sharing easy-to-understand notes.
* Continuously improve my technical and practical skills.
